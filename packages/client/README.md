# @u-devtools/client

Client-side UI shell for Universal DevTools. Provides the main DevTools interface rendered in an isolated iframe.

## Installation

```bash
npm install -D @u-devtools/client
```

## Usage

This package is automatically used by `@u-devtools/vite` plugin. You typically don't need to import it directly.
