# @u-devtools/overlay

Overlay UI component for Universal DevTools. Provides the floating launcher button and overlay menu.

## Installation

```bash
npm install -D @u-devtools/overlay
```

## Usage

This package is automatically used by `@u-devtools/vite` plugin. You typically don't need to import it directly.
